## Top Lessons

### What Killed Projects
1. No concrete recruitment plan (100%)
2. Massive overscope - 6-8 features instead of 3-4 (80%)
3. Unvalidated incentive assumptions (80%)
4. Technical red flags: mobile, ML, video, GIS (70%)
5. Two-sided marketplace cold-start (60%)

### What Would Save Them
1. Penn-only hyperlocal focus
2. 50-70% feature cuts
3. Week 1 validation before building
4. Named recruitment channels
5. Avoid rubric red flags
6. Manual MVP first
